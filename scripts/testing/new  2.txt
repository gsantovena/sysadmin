Deploy Process:

- SVN Update
- deploy-process.ps1 -Environment $environment
	- this will create a package file to deploy

.EXAMPLE
C:\PS> deploy.ps1 -Action Deploy -Datacenter haf1 -Webservers cpweb-a01-haf1,cpweb-a02-haf1 -App searchapi -PackageFileName haf.zip

It will deploy searchapi to the webservers cpweb-a01-haf1 and cpweb-a02-haf1 within haf1 datacenter from haf.zip package file.

.EXAMPLE
C:\PS> deploy.ps1 -Action Create -Datacenter admin -Webservers cpadmin-haf -App search -PackageFilename haf1-20120621.zip

It will create a package file named haf1-20120621.zip from cpadmin-haf webserver within admin datacenter.
